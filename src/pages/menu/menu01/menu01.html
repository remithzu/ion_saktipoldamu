Peta

klik icon pada menu tutama untuk membuka menu Peta.

di menu ini menampilkan Peta dari Google Map. tekan tombol berbentuk lingkaran pada pojok kanan atas untuk menampilkan menu lebih lanjut.

1 Untuk menampilkan peta berbasis ROADMAP.

2 Untuk menampilkan peta berbasis SATELLITE.

3 Untuk menampilkan daftar kantor polisi terdekat dengan anda.


Klik salah satu marker pada peta untuk memunculkan menu tersembunyi. "Menuju Lokasi" untuk membuat rute perjalanan


Klik tombol berbentuk lingkaran pada pojok kanan bawah untuk memulai navigasi.
